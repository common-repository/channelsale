== Installation ==

Manual installation involves downloading the plugin and uploading it to your webserver via your favorite FTP application. The WooCommerce codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

Steps listed below are shown in this demo video file (time length 98 seconds, file size 3 MB): [Demo Video](http://support.channelsale.com/WordPress%20_ChannelSale_Plugin_Demo_for_Testing_Team.mp4)

1. Upload the ChannelSale plugin using the add new plugin option. Once it is uploaded successfully, activate it.
2. Once activated, you will see a menu on the left side in WooCommerce admin named ChannelSale. Click on it.
3. Click on the New Customer Signup button. When you click on it, you will be redirected to https://www.channelsale.com/support/contact-us.aspx.
4. Fill in the username and password provided by ChannelSale. For testing, use:
   - Username: steve@channelsale.com
   - Password: sGhe42!we#
5. After entering the username and password, click save. You will get a message on successful connection with ChannelSale and a link to login to the ChannelSale account to proceed further.
